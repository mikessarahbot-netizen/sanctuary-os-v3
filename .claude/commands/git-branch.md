# /git-branch $MODULE $DESCRIPTION

Create and checkout a new feature branch before starting work.

Branch name: `feature/<module>-<description>` (lowercase, hyphenated)
Example: `feature/planning-setlist-generator`

1. Confirm you are branching from an up-to-date `main`.
2. Create and checkout the branch.
3. Report the branch name and confirm it is checked out.
